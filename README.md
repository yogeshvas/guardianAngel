## Run Locally
Clone the project

```bash
  git clone https://github.com/yogeshvas/kalesh.git
```

Go to the project directory

```bash
  cd kalesh
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx expo start
```
